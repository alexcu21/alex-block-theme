== Description ==

Alex's Block Theme is a modern WordPress block theme designed specifically for tech book clubs and learning communities. It features a clean, professional design with orange and teal accents, optimized for engagement and community building.

Key Features:
* Full Site Editing (FSE) support
* Modern block-based design
* Responsive layout that works on all devices
* Custom color palette with tech-focused colors
* Block patterns for easy content creation
* Custom typography with system fonts
* Accessibility-ready design
* Translation-ready
* WooCommerce compatible

Perfect for:
* Tech book clubs
* Learning communities
* Educational websites
* Technology blogs
* Community forums
* Online courses

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in 'Alex's Block Theme' in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance -> Editor (Beta) to customize your site using the Site Editor.

== Frequently Asked Questions ==

= Does this theme support the new Site Editor? =

Yes! Alex's Block Theme is built specifically for WordPress's Full Site Editing feature. You can customize every part of your site using the Site Editor.

= Is this theme responsive? =

Absolutely! The theme is designed to work perfectly on all devices, from mobile phones to desktop computers.

= Can I customize the colors? =

Yes, the theme includes a custom color palette that you can modify through the Site Editor's Global Styles panel.

= Does this theme support WooCommerce? =

Yes, the theme includes basic WooCommerce support for online stores.

= Is the theme translation-ready? =

Yes, the theme is fully translation-ready and compatible with WordPress language packs.
